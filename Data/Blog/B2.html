How to host site on github

9:33PM, 7th Aug 2021

If you want to host your site on github, this is the article you are looking for. I have given a step by step guide to host your site. Once you read this article carefully, believe me, hosting a site will take less time than the time it took to read this article.

Pre-Requirements:

You should have only one thing which is a github account. Create a new repository and this will be your third part of the site. And then upload all necessary files to your site here. Ensure that you have a file named index.html in that repository so that github will make it as your home page.
Example:
I have posted all my code in the "VipulNikam" repository and my site was “https://iam41.github.io/VipulNikam/”.

Make Website:

After uploading your code to your repository, you just have to go to setting > github pages and change setting from None to main. And your website url will appear on top of the page.

Open the link and confirm if there is no problem.
Viola your website is ready.
